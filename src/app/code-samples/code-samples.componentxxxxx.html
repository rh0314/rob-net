    rOrderBy Custom Angular pipe for sorting data - mostly usable in conjunction with *ngFor loops. Sorts any part of an array, whether it's only a single layer deep (i.e., an array of strings or integers) or if you need to sort an array of objects by a property that's multiple layers deep (i.e., student.class.school.district, if you were sorting a list of students by school district name.)
